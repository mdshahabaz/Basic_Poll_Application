# Basic_Poll_Application
#### This is a basic poll application, developed using Django Framework and Python as programming language.

* The application is divided into four segments for better understanding:
   * 1. Project, Views and Database setup
   * 2. Creating models, activating models and simulating Database APIs.
   * 3. Creating Admin side, Modifying views and Automated Testing Strategies.
   * 4. Django Test client, Customizing your app's look and feel.
